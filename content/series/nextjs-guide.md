# Next.js 완전정복 시리즈

Next.js는 React 기반의 풀스택 프레임워크로, 프로덕션 레벨의 웹 애플리케이션을 쉽게 만들 수 있도록 도와줍니다.

## 🎯 학습 목표

- Next.js의 **핵심 기능** 마스터
- **App Router** 완벽 이해
- **서버 컴포넌트**와 **클라이언트 컴포넌트** 구분
- **성능 최적화** 및 SEO 전략

## 📚 시리즈 구성

1. **Next.js로 블로그 만들기**
2. **App Router 심화 학습**
3. **데이터 페칭과 캐싱**
4. **인증과 보안**
5. **배포와 모니터링**

## 🚀 실습 프로젝트

- 개인 블로그 시스템
- E-commerce 플랫폼
- SaaS 대시보드
